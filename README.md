# Tienda TDEA
Clonar proyecto
npm install
Clonar el archivo .env.template y renombrarlo a .env
Cambiar las variables de entorno
Levantar la base de datos
docker compose up -d
Levantar: npm run start:dev

Ejecutar SEED

http://localhost:3000/api/seed
